# web-programming-project
1000명의 인생을 담은 블로그

> 인생의 전부를 담은 자서전 1000권을 읽으면 본인의 인생도 달라진다는 명언을 따라 만든 블로그
> > 1000권의 자서전을 추천해주는 블로그이다

## 페이지

`main` 페이지

`library` 페이지

`comment` 페이지

## 사용 언어
html, css, javascript 사용
